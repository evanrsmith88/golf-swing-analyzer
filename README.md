# Golf Swing Analyzer

A computer vision app that analyzes golf swing videos using pose estimation.

## Planned Features

- Upload a golf swing video
- Detect body landmarks using pose estimation
- Draw skeleton overlay on the video
- Calculate body and arm angles
- Compare swing positions to reference swings
- Give beginner-friendly feedback

## Tech Stack

- Frontend: Next.js, React, Tailwind
- Backend: Python, FastAPI
- Computer Vision: OpenCV, MediaPipe
